# Sci-Fi School Project Website

This is a mobile-friendly sci-fi themed website prepared for QR-code access.

## Files

- `index.html`: Main page structure with editable sections
- `styles.css`: Sci-fi visual theme and responsive layout
- `script.js`: Mobile menu behavior

## How to open locally

Open `index.html` directly in your browser, or use a local server.

Example with Python:

```bash
python -m http.server 8000
```

Then open: `http://localhost:8000`

## Publishing for QR code

To make the QR code work on phones, publish this website to a public URL (for example GitHub Pages, Netlify, or Vercel), then generate a QR code pointing to that URL.
